# Portfolio de Youssouph Sagna

Portfolio personnel présentant mon parcours en **Génie Logiciel**, mon **Master 1 en Sécurité des Systèmes d’Information (ESMT)**, ainsi que mes expériences à la **Banque Atlantique Sénégal** et mes différents projets, certifications et centres d’intérêt.

## Contenu du portfolio

- **Accueil** : présentation rapide, situation actuelle (CDD, Master 1 SSI) et accroche métier.  
- **Profil** : résumé de mon parcours, de mes objectifs et de ma vision.  
- **Compétences** : analyse des besoins, conception logicielle, développement (Java, JavaScript, Python, SQL, HTML, CSS), sécurité, méthodes de travail et outils (GLPI, Orion…).  
- **Expériences** : CDD Développeur & Support IT, stage à la Banque Atlantique, expériences associatives.  
- **Formations** : Licence 3 Génie Logiciel (Groupe ISI), Master 1 SSI (ESMT), Baccalauréat S2.  
- **Certifications** : Low Code / No Code, sécurité, data, gestion de projet, leadership, marketing digital, GLPI, front‑end, etc.  
- **Projets** : mémoire sur une plateforme de vente en ligne de récoltes agricoles.  
- **Contact** : e‑mails, téléphones, localisation et lien GitHub (`you7716`).  

## Technologies utilisées

- **HTML5**  
- **Tailwind CSS (CDN)**  
- Un peu de **JavaScript** pour :
  - le scroll fluide,
  - l’animation d’apparition des sections,
  - l’affichage automatique de l’année dans le pied de page.

## Comment l’ouvrir en local

1. Cloner ou télécharger ce dépôt.  
2. Ouvrir le dossier dans votre explorateur de fichiers.  
3. Double‑cliquer sur le fichier **`index.html`** pour l’ouvrir dans votre navigateur (Chrome, Edge, Firefox…).  

Le site est entièrement statique, il n’y a pas besoin de serveur ou d’installation particulière.

## Déploiement rapide sur GitHub Pages

1. Pousser ce projet sur un dépôt GitHub (par exemple `you7716/portfolioMoi`).  
2. Sur GitHub, aller dans **Settings → Pages**.  
3. Dans **Source**, choisir la branche **`main`** et le dossier **`/ (root)`**.  
4. Enregistrer : après quelques minutes, votre portfolio sera disponible à une adresse du type  
   `https://you7716.github.io/portfolioMoi/`.
